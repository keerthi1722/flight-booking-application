# Setup & Installation

## Prerequisites
- Node.js (version X.X.X or higher)
- npm or yarn

## Installation

### 1. Clone the repository
```
git clone <repo-url>
cd Flight Finder Rev
```

### 2. Install dependencies
```
cd client
npm install
cd ../server
npm install
```

## Running the Application

### Start the backend
```
cd server
npm start
```

### Start the frontend
```
cd client
npm start
```
